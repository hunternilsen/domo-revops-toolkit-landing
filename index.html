Setup

Takes about 5 minutes. The script handles everything.

1

Clone the repository

Get a copy of the toolkit on your machine.

git clone <repo-url> domo-revops-toolkit
cd domo-revops-toolkit
2

Run the setup script

This installs dependencies, configures your role, connects to the Knowledge Graph, Figma, and GitHub, and sets up daily auto-updates.

bash setup.sh
3

Start building

Open Claude Code and describe what you want. Try one of these:

# Open Claude Code
claude

# Then just say what you need:
"Build me a Custom App for tracking our sales pipeline"
"Create a Code Engine function that scores leads"
"Design a landing page for our new RevOps dashboard"

Commands

Type / in Claude Code to see these, or just describe what you want.

Build & Engineer

/build-custom-app Scaffold a complete Domo Custom App (Vite + React + TypeScript)
/build-code-engine Create a Domo Code Engine server-side function package
/build-workflow Design and generate a Domo Workflow automation

Design & Content

/design-landing-page Build a responsive landing page with Domo branding
/design-email-template Create a 600px responsive HTML email
/design-presentation Build an HTML slide deck with Domo styling
/create-icon Generate SVG icons in Domo's minimalist line style

Figma

/figma-to-code Convert a Figma design into real HTML/CSS
/figma-inspect Analyze a Figma file's components and brand compliance

Research

/query-revops Look up Domo products, platform capabilities, and engineering patterns
/get-product-info Deep-dive on any Domo product

Utilities

/new-project Create a new project with organized folders
/export-project Package project deliverables for sharing
/status Show toolkit health dashboard
/fix-my-setup Diagnose and repair broken connections
